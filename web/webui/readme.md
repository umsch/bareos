# WebUI

## Start
* start bareos-dir
* start webapi
* load webui into browser


Start Development server
```bash
npm i
npm run serve
``` 

## Used Components

|package|used for|
|---|---|
| axios | http client |
|buefy, bulma|web components|
|rxjs|event handling|
|vuex|app state|
|moment|date formating|
|lodash|swiss arm knife for arrays|
|mocha, chai|unit testing|
